# JWT_tester

For testing https://github.com/voscarmv/rails7_devise_jwt

# Installation

While running the rails7_devise_jwt server with `rails s`, do

```
$ cd JWT_tester
$ npm install
```

# Usage

```
$ node index.js
```
Copy the resulting Auth token and paste it where it belongs in login.js, then

```
$ node login.js
```
